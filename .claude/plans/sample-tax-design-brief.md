## 4. 強み & 差別化ポイント (クロ補填)

> ヒアリングで「適当に補填して」とのことで、以下を提案。
> 不動産オーナー特化 + 若手代表 + クラウド対応 という 3 つの軸で差別化。

### 差別化の 3 本柱

#### ① 「不動産オーナー専門」という明確な特化
- 一般税理士事務所は「業界横断・なんでも対応」
- みらい税理士事務所は **不動産特化** で、不動産特有の論点 (減価償却・修繕費 vs 資本的支出・消費税の課税事業者選択 等) に強い
- **数字**: 顧問先 80 社のうち 60 社 (75%) が不動産オーナー

#### ② 「個人 → 法人化」までワンストップサポート
- 賃貸経営が大きくなると **資産管理会社設立 (法人化)** を検討するフェーズが来る
- みらい税理士事務所は法人化の意思決定 → 会社設立 → 顧問契約までワンストップ
- **数字**: 法人化サポート実績 30 件以上

#### ③ クラウド完結 (全国対応 + 若手代表のスピード感)
- freee / マネーフォワード対応、書類は全部クラウド共有
- Zoom + チャット (Slack 或いは LINE 公式) で完結 → 来所不要
- **数字**: 平均レスポンス 24 時間以内、面談は最短翌日 Zoom

### 強調コピー候補
- 「不動産オーナーのための、クラウド完結税理士」
- 「個人 → 法人化までワンストップ。賃貸経営の節税は専門家へ。」
- 「不動産投資歴 10 年の若手税理士が、あなたの資産形成に伴走。」

### 6-5. Works (解決事例) — CMS 連動

匿名化した 3-6 件の事例:

```
事例 1: 「サラリーマン大家から法人化、3 年で年商 3 倍に」
        - 業種: 不動産賃貸 (区分 8 戸 → アパート 1 棟 + 区分 12 戸)
        - 課題: 個人で確定申告負担増 + 節税限界
        - 解決: 資産管理会社設立、相続対策込み
        - 結果: 法人税負担 30% 減、相続対策完了

事例 2: 「インボイス対応で 50 万円の節税」
        - 業種: 商業ビルオーナー
        - 課題: インボイス制度対応、消費税課税事業者の判断
        - 解決: 適格請求書発行事業者登録 + 仕入税額控除最適化
        - 結果: 年間 50 万円の節税

事例 3: 「相続税対策で次世代に資産を引き継ぐ」
        - 業種: 不動産オーナー (3 棟所有)
        - 課題: 相続発生時の税金が心配
        - 解決: 生前贈与計画 + 法人化 + 不動産小口化商品
        - 結果: 相続税予測額 4,000 万円 → 1,800 万円に圧縮
```

## 8. デザイントーン

### 配色 (ネイビー系・士業の王道)

```css
:root {
  /* メインパレット */
  --primary:      #1A2B4A;  /* 深いネイビー (信頼の核) */
  --primary-2:    #2C4068;  /* やや明るいネイビー (ホバー等) */
  --accent:       #C9A961;  /* くすんだゴールド (権威・上質感) */
  --bg:           #FFFFFF;  /* ホワイト基調 */
  --bg-2:         #F7F8FA;  /* 微かなブルーグレー (セクション alt 用) */
  --fg:           #1A2B4A;  /* 本文 = primary と同じ */
  --fg-2:         #4A5670;  /* 補助テキスト */
  --fg-3:         #8B97AB;  /* 弱補助 */
  --hairline:     #DCE2EB;  /* 罫線 */
}
```

**色の使い分け**:
- ネイビー: メイン CTA / 見出し / 重要要素
- ゴールド: アクセント (Hero の数字・差別化 No.01-03 の番号)
- ホワイト / 薄いブルーグレー: 背景の階層

### タイポグラフィ

```css
/* 和文: 信頼感のある明朝寄り or sans 高品質 */
--font-jp-sans: "Noto Sans JP", "Yu Gothic", sans-serif;  /* 本文 */
--font-jp-serif: "Noto Serif JP", "Yu Mincho", serif;     /* 見出し (明朝) */

/* 欧文: Inter or Montserrat */
--font-latin: Inter, -apple-system, sans-serif;

/* ロゴ・特別な強調: Caveat (filled-in) は使わない、士業らしくセリフ系 or Bebas Neue */
```

### グラフィック方向性

- **写真**: 建物のシルエット / 都市風景 / オフィスの抽象的な写真 (人物は控えめ、または使わない)
- **イラスト**: 使うとカジュアル過ぎる → アイコン (細線) のみ
- **アイコン**: Lucide React の細線アイコン or 自作 SVG (テンプレの Icons.tsx ベース)
- **モーション**: 控えめ、fade-in 程度。派手な animation は避ける

### レイアウト

- **余白**: 多め (clamp(80px, 12vw, 160px) のセクション余白)
- **罫線**: 細い (1px / hairline)、グレートーン
- **角丸**: ほぼ無し (border-radius: 2px 程度)、士業らしい誠実さ
- **影**: ほぼ無し or 極控えめ (box-shadow: 0 1px 3px rgba(26,43,74,0.06))
